JS -1 What's the difference between a variable that is: null, undefined or undeclared? and how would you go about checking for any of these states?

Ans:

"undefined" is a variable that has been declared but no value exists and is a type of itself ‘undefined’.

"null" is a value of a variable and is a type of object.

"undeclared" is a variable that has been declared without "var". In "use strict" mode, it would be occurred error.

I'll use "typeof" to check "undefined" or "null".

JS -2 Could you explain what’s the favour(difference) of Linked List and Array ?

Ans:

Linked List has extra node for pointing next node, and it is more flexible to change the order. 

  var test = ['a', 'b', 'c'];
  function linkNode(option) {
    this.value = option.value;
    this.next = option.next || null;
  }

  var node_a = new linkNode({value: 'a'}); 
  var node_b = new linkNode({value: 'b'});
  node_a.next = node_b;

  console.log(node_a.value + "," + node_a.next.value);
